Прогнозирование распространения COVID-19 в Беларуси

Участникам предстоит построить прогноз, наиболее точно отражающий динамику числа зарегистрированных случаев заболевания и смертности в Республике Беларусь.

Период прогнозирования: с 1 Июня 2020 по 1 Июля 2020.

Победитель за лучший прогноз получит вознаграждение в размере $100 (спонсоры приветствуются).

Участники могут использовать любые источники данных конечные прогнозы будут валидироваться по 2019 Novel Coronavirus COVID-19 (2019-nCoV) Data Repository by Johns Hopkins CSSE.

Формат прогнозов

Необходимо построить прогноз на все даты периода прогнозирования. Для каждой даты нужно предсказать количество зафиксированных случаев и смертей (всего по состоянию на этот день). Прогноз начинает действовать на следующий день после отправки прогноза, не учитывая предыдущие даты.Таблица лидеров будет обновляться сразу после обновления данных Johns Hopkins CSSE.

Данные необходимо отправить на странице Отправки прогнозов в формате .CSV

Пример .csv файла

date, prediction_confirmed, prediction_deaths
2020-06-13,45238,4256
2020-06-14,48985,4789
2020-06-15,50248,4977
2020-06-16,55123,5120

Метрика оценки прогнозов

  • PredictedConfirmed - прогнозируемое количество подтвержденных случаев на дату пересчета метрики;
  • ActualConfirmed - прогнозируемое количество подтвержденных случаев на дату пересчета метрики;
  • PredictedDeths - прогнозируемое количество смертей на дату пересчета метрики;
  • ActualDeths - прогнозируемое количество смертей на дату пересчета метрики;
  • d - количество дней прошедших с момента отправки прогноза.
